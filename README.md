# AgentJiraTicket

A Jira ticket agent project.

## Overview

This project contains AI agents for automating Jira ticket management.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

```bash
npm install
```

### Usage

```bash
npm start
```

## Project Structure

```
AgentJiraTicket/
├── .agentbase/
├── email-attachment-agent/
├── greennode-agentbase-skills/
└── jira-agent/
```
